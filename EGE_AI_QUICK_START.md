# 🚀 AI-проверка ЕГЭ/ОГЭ - Быстрый старт

## Для преподавателя: Как создать ДЗ с AI-проверкой ЕГЭ

### Шаг 1: Создайте новое ДЗ

1. Откройте **Конструктор ДЗ** (`/teacher/homework/new`)
2. Заполните основные поля:
   - Название: "Сочинение ЕГЭ - Задание 27"
   - Описание: "Напишите сочинение по прочитанному тексту"

### Шаг 2: Выберите тип экзамена

![Выбор типа](docs/screenshots/exam_type_selector.png)

```
Тип задания: [ЕГЭ (часть 2) ▼]
  ├─ Предмет: [Русский язык ▼]
  └─ Задание: [№27 - Сочинение ▼]
```

**Что происходит:**
- Система автоматически загрузит критерии ФИПИ для выбранного задания
- AI будет проверять работы строго по этим критериям

### Шаг 3: Добавьте вопрос

```
┌─────────────────────────────────────────────────┐
│ Тип вопроса: [Текстовый ответ (развернутый)]    │
│                                                 │
│ Текст задания:                                  │
│ ┌─────────────────────────────────────────────┐ │
│ │ Прочитайте текст и напишите по нему         │ │
│ │ сочинение. Сформулируйте проблему,          │ │
│ │ прокомментируйте её с двумя примерами...    │ │
│ └─────────────────────────────────────────────┘ │
│                                                 │
│ Баллы: [25]  (автоматически из критериев)      │
│                                                 │
│ ☑ Включить AI-проверку по критериям ФИПИ       │
└─────────────────────────────────────────────────┘
```

### Шаг 4: Опубликуйте ДЗ

Нажмите **"Опубликовать"** → ДЗ появится у учеников

## Часто задаваемые вопросы

### ❓ AI может ошибаться?

Да, AI - это помощник, а не замена учителю. Поэтому:
- ✅ Учитель **всегда** может изменить оценку
- ⚠️ Работы с низкой уверенностью AI (<75%) автоматически помечаются для ручной проверки
- 📊 Система учится на исправлениях учителя

### ❓ Как AI понимает критерии ФИПИ?

1. Критерии загружены из официальных документов ФИПИ
2. AI "видит" точные формулировки требований для каждого балла
3. Plus: в промпте есть 1-2 примера проверенных работ для калибровки

### ❓ Что если ученик списал?

AI **не детектирует** плагиат, но может заметить:
- Несоответствие стиля (слишком сложные конструкции)
- Копипасту из вопроса (пометит в warnings)

Для антиплагиата используйте отдельные сервисы.

### ❓ Можно ли использовать для других предметов?

**Уже доступно:**
- ✅ Русский язык (сочинение)
- ✅ Английский язык (эссе, письмо)
- ✅ Обществознание (понятие + примеры)
- ✅ История (аргументация)

**Планируется:**
- 🔄 Биология (развернутые ответы)
- 🔄 Химия (расчетные задачи с пояснениями)
- 🔄 Физика (задачи с обоснованием)

### ❓ Сколько это стоит школе?

**При 1000 проверок в месяц:**
- DeepSeek (рекомендуется): **~$0.10** (~7.5₽)
- С кешированием: **~$0.07** (~5₽)

**Для сравнения:**
- Средняя зарплата проверяющего: 400₽/час
- Время проверки 1 сочинения: ~15 минут
- Стоимость ручной проверки 1000 работ: **~100,000₽**

**Экономия: 99.99%** 🚀

### ❓ AI заменит учителей?

**Нет!** AI - это инструмент автоматизации рутины:
- ✅ **AI проверяет** формальные критерии (орфография, структура)
- 👩‍🏫 **Учитель даёт** персональные советы, мотивацию, развивает мышление
- 🤝 **Вместе** = быстрее проверка + больше времени на общение с учениками
