## 🎭 Анимации

### Shake (ошибка формы)

```
→ ← → ←  ← Встряхивание при ошибке входа
```

**Код:**
```css
@keyframes shake {
  0%, 100% { transform: translateX(0); }
  25% { transform: translateX(-10px); }
  75% { transform: translateX(10px); }
}
```

### Slide Up (появление)

```
    ↑
    │  ← Плавное появление снизу
┌───────┐
│       │
└───────┘
```

**Код:**
```css
@keyframes slideUp {
  from {
    opacity: 0;
    transform: translateY(30px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
```

### Slide Down (выпадающее меню)

```
┌─────────┐
│ Профиль │
└────┬────┘
     ↓
┌─────────────┐
│ Настройки   │
│ Выйти       │
└─────────────┘
```

## 📏 Отступы и размеры

### Spacing Scale

```
0.25rem = 4px   → --space-xs
0.5rem  = 8px   → --space-sm
0.75rem = 12px  → --space-md
1rem    = 16px  → --space-base
1.5rem  = 24px  → --space-lg
2rem    = 32px  → --space-xl
2.5rem  = 40px  → --space-2xl
3rem    = 48px  → --space-3xl
```

### Border Radius

```
8px  → --radius-sm   (кнопки, inputs)
12px → --radius-md   (карточки)
16px → --radius-lg   (большие секции)
```

### Font Sizes

```
0.85rem  → Мелкий текст (статистика)
0.9rem   → Обычный текст (формы)
0.95rem  → Средний текст (навигация)
1rem     → Базовый текст
1.125rem → Заголовки карточек
1.5rem   → Заголовки секций
2rem     → Заголовки страниц
```

## 💡 Советы по использованию

### 1. Консистентность

✅ **Правильно:**
- Используйте одинаковые отступы везде (1.5rem между секциями)
- Один стиль кнопок для одинаковых действий
- Единая анимация для hover эффектов

❌ **Неправильно:**
- Разные отступы в разных местах
- Смешивание стилей кнопок
- Разные скорости анимаций

### 2. Цвета

✅ **Правильно:**
- Primary (#2563eb) для основных действий
- Error (#ef4444) только для ошибок
- Success (#10b981) только для успеха

❌ **Неправильно:**
- Красный для обычных кнопок
- Зеленый для нейтральной информации
- Смешивание цветовых схем

### 3. Адаптивность

✅ **Правильно:**
- Тестируйте на всех размерах экрана
- Используйте относительные единицы (rem, %)
- Скрывайте текст на мобильных, оставляя иконки

❌ **Неправильно:**
- Фиксированные пиксели для всего
- Игнорирование мобильных устройств
- Одинаковый layout для всех экранов
